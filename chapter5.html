Chapter 5: A Simple Room

5.1 Create an Environment
As a first step we want to set up a room in which we can present our data later. We will keep it simple and create a room out of two walls and a floor. To begin, we will save our scene in the "Assets" folder and create a new GameObject.
1: click File → Save Scene → create a new folder "Scenes" → save the scene as “scene1” in the new folder

2: right-click on Hierarchy window → click Create Empty → right-click on "GameObject" → Rename → name it "Environment"

“Environment” is a so called Empty GameObject. You can't see it in the scene or game window. We just use it to organize our project. GameObjects can become a child of other GameObjects by attaching them in the Hierarchy. The child of a GameObject will get a local position and a local scale relative to its parent. We want to attach some children to the “Environment” object soon. We should reset the position and scale attribute of the parent to ensure that the children are not getting deformed by these attributes. Attributes of GameObjects can be changed in the Inspector.
3: click on “Environment” in the Hierarchy → open the Inspector → click the gear icon under Transform → Reset

Our room should get a floor. To structure our project, we want “Floor” to be a child of “Environment”
4: right click on Hierarchy window → 3D Object → Cube → rename the new GameObject "Floor"

5: drag and drop “Floor” in the Hierarchy onto “Environment”

6: left click on “Floor” in the Hierarchy → open the GameObject Inspector → set Transform Scale to X=10, Y=0.1, Z=10

The scene window should show us a white floor now. The two walls will be created in the same way.
7: create two 3D Objects Cube → rename them to "Wall1" and "Wall2" → make them a child of "Environment"

8: open “Wall1” in the Inspector → set Transform Scale to X=10, Y=4, Z=0.1 → set the Transform Position to X=0, Y=2, Z=5
open “Wall2” in the Inspector → set Transform Scale to X=0.1, Y=4, Z=10 → set the Transform Position to X=-5, Y=2, Z=0

We should paint the floor and the walls to make the room look more realistic. Therefore we can create some materials and add them as a component to our GameObjects.
9: open the Assets folder in the Project explorer → right click in the Assets window → Create → Folder → name it “Materials”
10: go into the Materials folder → right click → Create → Material → name it “floor_material”

11: open “floor_material” in the Inspector → click on the Main Maps Albedo color icon → choose a color → drag and drop “floor_material” from the project explorer onto the floor in the scene window

12: create a new material → name it “wall_material” → change its color → drag and drop it onto the walls
