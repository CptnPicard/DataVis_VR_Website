6. The Coordinate System

6.1 The Coordinate System
In this section we are going to implement a simple coordinate system into the project. The coordinate system should have a XY-, XZ- and YZ-Plane. The edges between these planes will represent the x-, y- and z-axis. We will create the coordinate system similar to the simple room, but this time we will use a transparent material.
1: create an empty GameObject → rename it “Coordinate System” → set the Transform position to X=0, Y=1.5, Z=0
2: create three 3D GameObjects Cube → rename them “XY-Plane”,”XZ-Plane”-”YZ-Plane” → let them become children of “Coordinate System”
3: change the Transform Position for the XY-Plane to X=0, Y=0, Z=0.5 → change Transform Scale to X=1, Y=1, Z=0.01
4: change the Transform Position for the XZ-Plane to X=0, Y=-0.5, Z=0 → change Transform Scale to X=1, Y=0.01, Z=1
5: change the Transform Position for the YZ-Plane to X=-0.5, Y=0, Z=0 → change Transform Scale to X=0.01, Y=1, Z=1
6: go to the project explorer → open the "Materials" Folder → create a new Material → rename it “coordinate_material" → open the Shader dropdown menu in the Inspector → select Legacy Shaders/Transparent/Diffuse → choose a color and set the A-value to 50 → drag and drop the material onto the three planes
