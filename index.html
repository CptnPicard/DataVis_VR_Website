Visualize your data in Virtual Reality!

This website will motivate the visualisation of data in Virtual Reality in the context of Data Science. Furthermore it contains a tutorial on how to visualize data in Virtual Reality using the development environment Unity. The tutorial is provides a step by step instruction to reconstruct the “DataVis VR” project. The tutorial doesn’t require any prior knowledge but basics in the programming language C# are recommended. The content of the website is splitted into 12 chapters. The first chapters are providing a introduction into Virtual Reality and Data Science. The tutorial for the “DataVis VR” project starts at chapter 4.
